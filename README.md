# TECH 5 STAR API

## Description

[Nestjs](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

First, copy env file:

```bash
cp env.example .env
```

Edit environment variable to your configuration

```bash
$ yarn
```

## Running the app

```bash
# watch mode
$ yarn start:dev

# build for production
$ yarn build

# start production mode
$ yarn start:prod
```
